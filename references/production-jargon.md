# 模块6：媒体行业行话词典

## 拍摄行话

| 术语 | 释义 | 使用场景 |
|------|------|---------|
| A-roll | 主要采访画面，核心叙事素材 | "这段A-roll表情很好，保留" |
| B-roll | 辅助画面，环境交代、情绪渲染 | "这里需要补B-roll，画面太空" |
| 空镜 | 无人物出现的画面 | "开场用3秒城市空镜" |
| 长镜头 | 不剪辑的连续拍摄画面 | "这段用长镜头更有沉浸感" |
| 收声 | 现场录音 | "今天收声环境不好，有噪声" |
| 走位 | 拍摄前确认人物位置和移动路线 | "先走一遍位" |
| 打板 | 拍摄开始前的场记板动作 | "打板，第三场第一镜" |
| 穿帮 | 画面中出现不该出现的内容 | "这个镜头穿帮了，背景有手机" |
| 灯位 | 灯光布置位置 | "主灯位调一下，脸上阴影太重" |

## 剪辑行话

| 术语 | 释义 | 使用场景 |
|------|------|---------|
| 粗剪 | 按脚本顺序初步剪辑 | "先出粗剪看节奏" |
| 精剪 | 细化节奏、转场、细节 | "精剪时注意音乐卡点" |
| 卡点 | 画面切换与音乐节拍同步 | "这个转场要卡点" |
| J-cut | 音频先于画面切入 | "这里用J-cut更自然" |
| L-cut | 画面先于音频切换 | "采访衔接用L-cut" |
| 蒙太奇 | 通过画面并列产生新意义 | "结尾用蒙太奇收束情绪" |
| 调色 | 统一画面色彩风格 | "整体调色偏暖，情绪向" |
| 达芬奇 | 达芬奇调色软件（行业标配） | "调色交给达芬奇" |
| 转场 | 场景之间的过渡方式 | "这里用什么转场？硬切还是叠化？" |
| 包装 | 字幕、片头、角标等视觉元素 | "包装还没上" |

## 运营行话

| 术语 | 释义 | 使用场景 |
|------|------|---------|
| 完播率 | 视频被完整看完的比例 | "完播率只有30%，3秒钩子要改" |
| 转化率 | 用户完成目标行为的比例 | "这条视频转化率不行" |
| 流量池 | 平台推荐系统的分层机制 | "卡在500流量池上不去" |
| DOU+ | 抖音付费推广工具 | "投DOU+测试数据" |
| 种草 | 让用户产生购买/行动意愿 | "这条笔记种草力不够" |
| 网感 | 对网络流行趋势的敏感度 | "这个标题网感不好" |
| 社交货币 | 能让人愿意分享的内容价值 | "这个话题没有社交货币属性" |
| 冷启动 | 内容发布初期的流量获取 | "冷启动数据决定后续推荐" |
| 钩子 | 吸引注意力的开场设计 | "前3秒的钩子不够强" |
| 爆款 | 远超平均表现的爆款内容 | "上个月出了2条爆款" |

## 审核行话

| 术语 | 释义 | 使用场景 |
|------|------|---------|
| 红线 | 不可触碰的审核底线 | "这个点踩红线了" |
| 亮红牌 | 直接否决 | "这个标题我亮红牌" |
| 擦边 | 接近但未明确触碰红线 | "这个算不算擦边？" |
| 导向 | 内容的政治方向和价值取向 | "导向没问题，可以发" |
| 把关 | 内容审核和风险控制 | "把关过了再发" |
| 一票否决 | 发现一个问题即全盘否决 | "政治导向一票否决" |
| 退稿 | 审核不通过退回修改 | "被退稿了，标题要改" |
| 送审 | 提交审核 | "先送审，别急" |

## 通用行话

| 术语 | 释义 | 使用场景 |
|------|------|---------|
| 选题 | 新闻/节目的主题选择 | "今天的选题会定了3个题" |
| 跑口 | 记者负责报道的领域 | "我跑政法口" |
| 通稿 | 统一发布的新闻稿 | "等通稿出来再跟进" |
| 策划 | 内容选题和执行方案 | "这个选题的策划还没过" |
| 落地 | 方案实际执行 | "这个策划能落地吗？" |
| 上会 | 提交到编辑会议上讨论 | "先上会看看" |
| 压稿 | 暂缓发布 | "这条稿先压一压" |
| 追热点 | 快速跟进热门话题 | "追热点要快，4小时窗口期" |
| 全案 | 完整的整合传播方案 | "做个全案，别只做单条" |
